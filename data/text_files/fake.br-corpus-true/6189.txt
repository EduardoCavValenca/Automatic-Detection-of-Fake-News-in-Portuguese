Câmara aprova projeto que cria banco de dados sobre violência contra a mulher. Objetivo da proposta é integrar sistemas de informações de instituições que atendem mulheres vítimas de violência. Texto segue para nova análise do Senado..  A Câmara dos Deputados aprovou nesta terça-feira (20) o projeto que cria a Política Nacional de Dados e Informações Relacionadas à Violência Contra as Mulheres. O objetivo da proposta é integrar os sistemas de informações de instituições que atendem mulheres vítimas de violência. O texto já havia sido aprovado pelos senadores, mas, como foi modificado pelos deputados, seguirá para nova análise do Senado. O objetivo do projeto é reunir, organizar e apresentar os dados sobre todos os tipos de violência contra as mulheres. Para isso, prevê a integração das bases de dados das instituições que atendem as mulheres em situação de violência. A proposta define que será criado o Registro Unificado de Dados e Informações sobre a Violência Contra as Mulheres. O cadastro deverá conter alguns dados, entre os quais: Um comitê vai acompanhar a implantação da Política Nacional de Dados e Informações Relacionadas à Violência Contra as Mulheres. O comitê terá representantes dos poderes Executivo, Legislativo e Judiciário. Depois de aprovar o projeto, a Câmara aprovou uma outra proposta, que altera as regras do processo de fiscalização dos produtos comestíveis de origem animal, produzidos de forma artesanal. O texto permite que estes produtos alimentícios de origem artesanal possam ser vendidos entre os estados, com fiscalização a cargo dos governos dos estados e do Distrito Federal. Um selo único, chamado ARTE, vai identificar as mercadorias. A intenção é desburocratizar os procedimentos para esta comercialização.
